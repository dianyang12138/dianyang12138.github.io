安装配置虚拟机环境

manjaro快速安装centos虚拟机

vagrant命令失败请检查是否在对应目录下，一般是家目录。

  1. uname -r查看版本号，5.7.17-2-MANJARO表示57

  2. sudo pacman -S virtualbox选择对应模块，这里是57就是linux57-virtualbox-host-modules

  3. sudo pacman -Ss virtualbox-ext-oracle安装扩展包

  4. sudo gpasswd -a 用户名 vboxusers：添加当前用户到vboxusers

  5. sudo modprobe vboxdrv：

  6. sudo pacman -S vagrant

  7. 重启

  8. vagrant：验证是否安装成功

  9. vagrant init centos/7：centos/7是仓库里名字对应，会在~下创建vagrantfile

  10. vagrant up：启动虚拟机，会下载并启动

    • 网络问题可能会下载超时，我采用的是把下载的地址在迅雷下载，然后把box文件用vagrant box add centos/7 ~/virtualbox.box添加，然后vagrant up

  11. vagrant ssh：可以直接连接虚拟机

    • exit：退出

    • #也可以根据信息连接
default: SSH address: 127.0.0.1:2222
default: SSH username: vagrant
default: SSH auth method: private key

离线安装idea插件：

下载，解压，放到/home/guanhe/program下。激活码

插件下载网址（下载的版本，最好下和自己idea版本对应的），选择settings，选择install from disks

安装VsCode

下载，解压，放到/home/guanhe/program下。
